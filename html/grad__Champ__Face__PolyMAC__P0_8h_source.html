46//! class grad_Champ_Face_PolyMAC_P0 for the calculation of the gradient<br>This field is a Champ_Fonc_Face_PolyMAC_P0 that calculates the gradient of a velocity field<br>grad_u(f, n*D + dU) returns the gradient of the dU velocity component in the phase n at face f<br>grad_u(nf_tot + e*D + dX, n*D + dU) returns the gradient of the dU velocity component in the phase n in element e along the dX component<br>(i.e. = dU/dX)<br>In 2D, the gradient at the element e in phase n is equal to :<br>| grad_u(nf_tot + e*D + 0, n*D + 0) grad_u(nf_tot + e*D + 1, n*D + 0) |<br>grad_u = | |<br>| grad_u(nf_tot + e*D + 0, n*D + 1) grad_u(nf_tot + e*D + 1, n*D + 1) |<br>The gradient is calculated using past values of the velocity<br>We compute the gradient at the faces for all velocity components,<br>then interpolate it to obtain the gradient at the elements
