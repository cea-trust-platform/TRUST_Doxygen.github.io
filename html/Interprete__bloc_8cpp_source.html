69//! Interpretation d'un bloc d'instructions prises dans l'entree is.<br>Si le bloc commence par {, on suppose que l'accolade a deja ete lue.<br>Le bloc se termine soit par }, soit par la fin du fichier, soit par<br>le mot FIN en fonction de bloc_type (voir mon_main.cpp par exemple)<br>Si verifie_sans_interpreter!=0, on ne cree aucun objet et on ne<br>lance aucun interprete, on se contente de verifier la coherence des<br>accolades (meme nombre de { que de }).
70Entree& Interprete_bloc::interpreter_bloc(Entree& is,
71 Bloc_Type bloc_type,
72 int verifie_sans_interpreter)
73{
74// Le niveau de sortie des messages journal
75 const int jlevel = 3;
76 Journal(jlevel) << "Interprete_bloc::interpreter_bloc bloc_type=" << (int)bloc_type << finl;
77 Motcle motlu;
78 is >> motlu;
79 while(1)
80 {
81// Est-ce qu'on est a la fin du bloc ?
82 if (is.eof())
83 {
84// Certains fichiers .data ne se terminent pas par FIN, on accepte aussi d'arriver a eof
85// sans avoir trouve "FIN".
86 if (bloc_type == BLOC_EOF || bloc_type == FIN)
87 {
88 Journal(jlevel) << "Interprete_bloc: end of file => end of bloc" << finl;
89 break;
90 }
91 else
92 {
93 Cerr << "Error in Interprete_bloc: unexpected end of file\n"
94 << "check for missig \"}\" or missing FIN keyword"
95 << finl;
96 exit();
97 }
98 }
99// Autres fins possibles:
100 if (motlu == "}" || motlu == "FIN|END")
101 {
102 if ((motlu == "}" && bloc_type == ACCOLADE)
103 || (motlu == "FIN|END" && bloc_type == FIN))
104 {
105 Journal(jlevel) << "Interprete_bloc: reading " << motlu << " => end of bloc" << finl;
106 break;
107 }
108 else
109 {
110 if (motlu == "}")
111 Cerr << "Error in Interprete_bloc: extra \"}\" in data file" << finl;
112 else
113 Cerr << "Error in Interprete_bloc: unexpected FIN, check for missing \"}\""
114 << finl;
115 exit();
116 }
117 }
118
119// Est-ce que c'est un commentaire ?
120 if (motlu == "#")
121 {
122 Nom commentaire("# ");
123 Nom nom_lu;
124 int countleft = 8; //!< On retient les 8 premiers mots du commentaire
125 do
126 {
127 is >> nom_lu;
128 if (is.eof())
129 break;
130 if ((countleft--) > 0)
131 {
132 commentaire += nom_lu;
133 commentaire += " ";
134 }
135 }
136 while (nom_lu != "#");
137 if (is.eof())
138 {
139 Cerr << "Error in Interprete_bloc: end of file while reading a comment :\n"
140 << commentaire << "..." << finl;
141 }
142// Prochain mot a interpreter:
143 motlu = nom_lu;
144 }
145 else if (motlu == "{")
146 {
147 Journal(jlevel) << "Interprete_bloc: reading { => creating new Interprete_bloc"
148 << finl;
149// Est-ce que c'est un debut de bloc ?
150// Nouvel interprete:
151 Interprete_bloc inter;
152 inter.interpreter_bloc(is, ACCOLADE, verifie_sans_interpreter);
153 }
154 else
155 {
156 if (verifie_sans_interpreter)
157 {
158// Si on veut seulement verifier le nombre d'accolades,
159// il suffit d'ignorer le motcle... on ouvre recursivement
160// de nouveaux interprete si on rencontre { et on les ferme
161// si on trouve }. S'il manque une } a la fin, on obtiendra le message
162// "check for missing }" et s'il y en a une en trop on aura
163// le message "extra }"
164 Journal(jlevel) << "Interprete_bloc: just checking {} => ignore keyword "
165 << motlu << finl;
166 verifie(motlu);
167 }
168 else
169 {
170// Dans le bloc suivant, s'il y a une erreur de lecture sur is, c'est une erreur:
171 is.set_error_action(Entree::ERROR_EXIT);
172 int export_object = 0;
173 if (motlu == "export")
174 {
175 Journal(jlevel) << "Exporting object, reading object type..." << finl;
176 export_object = 1;
177 is >> motlu;
178 }
179// Le mot cle doit etre un type d'objet instanciable
180 Journal(jlevel) << "Interprete_bloc: reading " << motlu
181 << " => trying to instanciate object" << finl;
182 DerObjU objet;
183 objet.typer(motlu);
184
185 if (sub_type(Interprete, objet.valeur()))
186 {
187// Si c'est un interprete, on appelle la methode interprete de l'objet:
188 Journal(jlevel) << " Calling object.interpreter()" << finl;
189 Interprete& inter = ref_cast(Interprete, objet.valeur());
190 inter.interpreter(is);
191 }
192 else
193 {
194// Ce n'est pas un interprete, on lit le nom de l'objet et on le stocke
195 Journal(jlevel) << " Reading object name" << finl;
196 Nom nom_objet;
197 is >> nom_objet;
198 Journal(jlevel) << " Storing object " << nom_objet << " of type " << motlu << finl;
199 if (!export_object || !pere_.non_nul())
200 ajouter(nom_objet, objet);
201 else
202 pere_.valeur().ajouter(nom_objet, objet);
203 }
204 is.set_error_action(Entree::ERROR_CONTINUE);
205 }
206 }
207 is >> motlu;
208 }
209 return is;
210}

236//! Ajoute l'objet ob a la liste des objets de l'interprete,<br>et nomme l'objet avec nom. Si l'objet existe deja, exit()
237Objet_U& Interprete_bloc::ajouter(const Nom& nom, DerObjU& ob)
238{
239 Journal(3) << "Interprete::ajouter(" << nom << ") de type "
240 << ob.valeur().que_suis_je() << finl;
241 if (les_noms_.search(nom) >= 0)
242 {
243 Cerr << "Error in Interprete::ajouter: object " << nom << " already exists." << finl;
244 exit();
245 }
246 les_noms_.add(nom);
247 Objet_U& obu = add_deplace(ob);
248 obu.nommer(nom);
249 return obu;

252//
253//
254//! cherche l'objet demande dans l'Interprete_bloc<br>courant (Interprete_bloc::interprete_courant()) et dans tous<br>ses peres successifs. Si l'objet n'existe pas, exit()
255Objet_U& Interprete_bloc::objet_global(const Nom& nom)
256{
257 REF(Interprete_bloc) ptr(interprete_courant());
258 while (ptr.non_nul())
259 {
260 Interprete_bloc& interp = ptr.valeur();
261 if (interp.objet_local_existant(nom))
262 {
263 Objet_U& objet = interp.objet_local(nom);
264 return objet;
265 }
266 ptr = interp.pere_;
267 }
268// Objet non trouve !
269 Cerr << "Error in Interprete::objet: object " << nom
270 << " does not exist." << finl;
271 Process::exit();
272 return objet_global(nom); //!< Pour le compilo
273}
