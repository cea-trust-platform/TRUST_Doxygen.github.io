35//! :<br>Cette classe decrit un groupe de processeurs sur lesquels<br>une portion de code s'execute simultanement. Elle fournit<br>toutes les methodes permettant d'echanger des donnees entre<br>les processeurs du groupe (mpsum, send, recv, ...),<br>et de synchroniser les processeurs (barrier).<br>Elle est specialisee selon le support reseau (MPI, PVM, ...)<br>Attention, ces methodes sont reservees a des operations de bas niveau<br>(noyau TRUST).<br>Dans le code courant, il faut utiliser les methodes classes de communications<br>de haut niveau :<br>(envoyer(), envoyer_broadcast(), class Schema_Comm, class Process, etc)<br>Pour creer un nouveau groupe et l'utiliser, voir class PE_Groups<br>Pour la procedure d'initialisation, voir PE_Groups::Initialize()
